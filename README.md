# DamageCalc α

> ⚠️ **Alpha版**  
> 開発中

---

## 概要

DamageCalc は、キャプチャ映像から相手パーティを検出し、登録した自分のパーティとのダメージ計算を補助する Windows 向けツールです。

ダメージ計算エンジンには **Smogon `@smogon/calc`** を使用しています。

---

| Python | 不要 |

| モニター環境 | デュアルディスプレイ以上 |

## セットアップ

※ビルド版（EXE配布版）では、この手順は不要です。

```bash
# Python依存パッケージのインストール
pip install -r requirements.txt

# Smogonダメージ計算エンジンのインストール
cd src/calc
npm install
cd ../..

# スプライトのDL
python -m src.data.download_champions_sprites

# 起動
python main.py
```

## 基本的な使い方

### 1. カメラを接続する

- 起動後、オプションの **カメラ** ドロップダウンからキャプチャデバイスを選択します。
- ゲーム映像が右ペインに表示されれば準備完了です。
 
### 解像度について
- Nintendo Switch 2 を使用する場合は、本体の出力設定、またはOBS仮想カメラ側の設定で **720p** にしてください。  
- 現在の自動認識処理は720p入力を前提にしています。

### 2. 自分のパーティを登録する

1. **「＋新規登録」** または **「ボックスから読み込む」** から使用ポケモンを登録します。
2. 登録したポケモンはリストに表示されます。
3. ドラッグ＆ドロップまたは左クリックで **自分PT** のスロットにセットします。
4. 右クリックメニューから情報を編集できます。

### 3. 相手のパーティを設定する

#### 方法A：自動検出

1. ゲームの **選出画面** を表示します。
2. **「相手PT検出」** を押すと、画面を解析して相手の6体を自動入力します。
3. 検出できなかったスロットは空欄になります。必要に応じて手動で補完してください。

#### 方法B：自動トリガー

- **「自動検出 ON」** をオンにすると、選出画面が表示されたタイミングで自動的に検出します。

#### 方法C：手動入力

- 相手PTの各スロットをクリックして、ポケモン名を選択します。

### 4. ダメージを計算する

1. 自分PT / 相手PT のスロットをクリックします。
2. ダメージ範囲と確定数が自動表示されます。
3. 表示結果を確認し、必要に応じて個体値・努力値・持ち物・技などを調整してください。

## 注意事項

- **EV（努力値）はチャンピオンズ仕様**です。アプリ内では「ポイント数」（従来の EV ÷ 8 相当）で入力します。
- 相手ポケモンの個体値・努力値は、使用率情報から計算された初期値が設定されます。
- 自動検出の精度は照明・キャプチャ品質・解像度によって変わります。検出結果は必ず確認してください。
- Alpha版のため、一部のポケモン・フォルムでは認識精度が低い場合があります（未検証のスプライトが含まれます）。
- 本ツールの計算結果は補助情報です。実際の対戦結果を保証するものではありません。

## トラブルシューティング

### カメラが映らない

OBS の仮想カメラが起動しているか確認してください。  
カメラデバイスの選択先を切り替えると改善する場合があります。

### ダメージ計算が動かない

開発版は `src/calc/` で `npm install` が完了しているか確認してください。  
ビルド版は Node.js がインストールされているか確認してください。

### 相手PT検出がうまくいかない

選出画面が完全に表示された状態で実行してください。  
画面の明るさ・解像度・キャプチャ品質の影響を受けることがあります。

### アプリが起動しない

`%USERPROFILE%\.pokemon_damage_calc\error.log` にエラーログが保存されています。  
エラー内容を確認してください。

### ダメージ計算エラーを報告したい

次の手順でログを取得して提出してください。

1. オプションを開き、**「詳細ログ」** にチェックを入れる
2. エラーが発生する操作を再現する
3. オプション内のログ出力機能でログを保存する
4. 保存したログを添付して報告する
